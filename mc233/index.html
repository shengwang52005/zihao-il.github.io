    Minecraft for Android 版本库¶

    这是一个的第三方的资源平台，我们持续给 网上各大版本库 提供安装包(Android)，同时也是全网安装包(Android)供应的源头之一

    本站地址
    • zihao-il.github.io
    • mc233.endyun.ltd（推荐）
    • bbk.endyun.ltd（动态）
    • mc233.endyun.ltd/old（旧版）
    • 版本库APP下载
    注意
    • 本站所有的安装包仅供交流学习，禁止分享到任何平台，下载后请24小时内及时删除！！！
    • 如需要游玩请前往Minecraft官网购买正版！！！
    • 本站的所有安装包均属于基岩版
    • 本站与 Mojang Studio 和 Microsoft 没有从属关系

    API¶

    演示站：bbk.endyun.ltd(移动端)

    http://bbk.endyun.ltd:9000/api

    POST: http://bbk.endyun.ltd:9000/api/last_version
data = {"b": "1"}
    参数 可选 说明
    b 是 是否为测试版(不填：获取全部，1：测试版，0：正式版) 
    http://bbk.endyun.ltd:9000/api/search_version?s=1.20.1&m=1&b=2
    参数 可选 说明
    s 否 查询的版本号
    m 是 是否启用模糊查询(1：启用)
    b 是 是否为测试版(不填/2：获取全部，1：测试版，0：正式版) 
    POST: http://bbk.endyun.ltd:9000/api/get_version
data = {"v": "1.20.x", "b": "1"}
    参数 可选 说明
    v 否 查询的大版本号(例如：1.20.x)
    b 是 是否为测试版(不填：获取全部，1：测试版，0：正式版)
